Έχουμε μια επαναλληπτική διαδικασία που σταματάει μόνο αν το επιλέξουμε εμείς.Μέσα εκεί γίνεται η κλήση της συνάρτησης που διαβάζουμε το ινπουτ που δίνει ο χρήστης.Μέσα σε αυτή τη συνάρτηση
αφου διαβάζει γίνεται έλεγχος για το τι τύπος εντολής είναι(απλη,συνθετη,pipe,redirection) και επίσης καθαρίζει το input απο κενα και ότι άλλο χρειάζεται να χειριστούμε.Μετά χειρίζομαι τις
περιπτώσεις exit και cd.Μετά παίρνουμε όλες τις περιπτώσεις για όλους τους τυπούς εντολών καλώντας την αντιστοιχη συνάρτηση.Η κάθε συνάρτηση χειρίζεται με τον αντιστοιχο τρόπο τη καθε περίπτωση
και καλύπτω όλες τις περιπτώσεις ακόμα και τα errors.
